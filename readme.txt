WPLife Switcher for WooCommerce поможет переключить, изменить, настроить многое в плагине WooCommerce

== Description ==

Плагин умеет:

Все настройки можно опционально включать или выключать в настройках плагина.

1. Добавить вариант сортировки: "По названию А-Я"
2. Добавить вариант сортировки: "По названию Я-А"
3. Удалить вариант сортировки: "по дате"
4. Добавить вариант сортировки: "По актикул"
5. Разрешить показывать пустые категории в магазине

== Installation ==

1. Распакуйте файл плагина и извлеките папку.
2. Загрузите папку wplife-woo-s в каталог /wp-content/plugins/.
3. Убедитесь что плагин WooCommerce активирован.
4. Активируйте плагин через меню "Плагины" в WordPress.

= Usage =

1. Перейдите в раздел "Switcher" в меню "Настройки".
2. Выберите варианты по вашему выбору
3. Нажмите кнопку "Сохранить изменения"
